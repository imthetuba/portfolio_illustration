Step-by-Step Instructions
Install Python:

Go to the Python website.
Download the latest version of Python for Windows.
Run the installer and make sure to check the box that says "Add Python to PATH" before clicking "Install Now".
Install Git (Optional but recommended):

Go to the Git website.
Download the latest version of Git for Windows.
Run the installer and follow the default settings.
Open Command Prompt:

Press Win + R, type cmd, and press Enter.
Verify Python Installation:

In the Command Prompt, type: python --version
You should see the Python version you installed. If not, ensure Python is added to your PATH.
Install Required Python Packages:

Additional Notes
Infront API Access:

Ensure you have an Infront account and API access. You will need your Infront username and password to use the tool.
CSV File:

Make sure the assets_indices_map.csv file is in the same directory as illustrationtool.py.
